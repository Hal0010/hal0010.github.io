flow matching

流匹配

流匹配是一种概率生成模型，通过学习一个常微分方程，实现从先验分布向目标分布的变换。 微分方程可以定义为： $$ \frac{d}{dt}x_{t}=v_{t}(x_{t}) $$

其中 vt(xt) 是时间和状态相关的速度场，这个速度场决定了数据如何从 p0 转换为 p1 。如果从 t = 0 的一个噪声样本开始，沿着这个向量场积分到 t = 1，就能得到一个数据样本 x1 ≈ p1。

由于无法得知整个分布的演化，观察一个特定的数据点 x1 ∼ p1 是如何从噪声演变来的。可以定义一个任意从噪声点 x0 ∼ p0 到这个特定数据点 x1 的路径。这个路径只依赖于选定的 x0和x1 ，记条件概率路径为 pt(x|x0, x1) ，定义了概率路径后就可以计算出对应的条件向量场ut(x|x0, x1) 。

eg:矫正流 使用线性插值定义概率路径： xt = (1 − t) ⋅ x0 + x1 ⋅ t 对时间求导：得到目标向量场： ut(xt|x0, x1) = x1 − x0 条件流匹配（CFM）的损失可以定义为： ℒCFM(θ) = 𝔼t, x0, x1[||vθ(xt, t) − ut(xt|x0, x1)||2] 其中，𝔼 表示对时间 t ∼ U(0, 1) ，噪声 x0 ∼ p0 和目标分布 x1 ∼ p1 进行采样求期望，实际计算形式为： ℒRectifiedFlow(θ) = 𝔼t, x0, x1[||vθ((1 − t)x0 + tx1, t) − (x1 − x0)||2] 数学上可以严格证明： 最小化条件匹配流损失 ℒCFM 等价于最小化边缘匹配流损失 ℒmarginal 故： ∇θℒCFM(θ) = ∇θℒmarginal(θ) 通过让模型学习 ut(xt|x0, x1)，即可得到目标向量场。
